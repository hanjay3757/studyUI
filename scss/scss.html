변수선언, 기본은 css SCSS(Sassy CSS)는 CSS의 확장 문법으로, CSS를 더 효율적이고 구조적으로 작성할 수 있게 도와주는 전처리기(preprocessor)입니다. SCSS는 기존 CSS 문법과 거의 동일하게 사용할 수 있으면서, 다음과 같은 추가 기능을 제공합니다. 주요 특징 변수(Variables) 색상, 폰트 크기 등 반복되는 값을 변수로 저장해 재사용할 수 있습니다. 중첩(Nesting) 선택자 안에 또 다른 선택자를 중첩해, 계층 구조를 명확하게 표현할 수 있습니다. 믹스인(Mixins) 자주 사용하는 스타일 묶음을 함수처럼 만들어 재사용할 수 있습니다. 상속(Extend) 기존 스타일을 다른 선택자에 상속해 코드 중복을 줄일 수 있습니다. 연산(Operations) 색상, 크기 등에서 사칙연산을 사용할 수 있습니다. 예시 요약
